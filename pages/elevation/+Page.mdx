# Elevation & depth

<Lede>
  Four shadow tints layered onto the panel surface. Light mode uses ink-tinted black at
  low opacity; dark mode uses pure black at higher opacity (the same custom property
  tracks both themes).
</Lede>

## Shadow scale

`tint-1` is the resting panel base; `tint-2` lifts a popover off the page; `tint-3`
floats a modal or vim-help overlay. The inset variant marks pressed states and focused
inputs.

<ShadowSpecimen />

## Z-index layers

A small fixed set of layers — never invent ad-hoc z-indices.

| Layer | z-index | What sits there |
|---|---|---|
| Page chrome (sticky topbar) | `50` | The doc topbar |
| Sticky in-page chrome | `3` | Sticky `<H2>` headings inside the prose |
| Page content | `auto` (0) | Default — everything in the doc body |
| Overlay backdrop | `70` | Dim layer behind the search palette |
| Overlay surface | `71` | The search palette + input itself when open |
