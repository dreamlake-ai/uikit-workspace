## With a name

The avatar atom itself is just the square — no built-in name slot. When a
display name is needed (sidebar identity, workspace switcher, header
rows), compose `<Avatar />` with a UI-font label in a flex pair: 9px gap,
12.5px ink text, no second weight or color shift. Keep the avatar as the
visual anchor; the name reads as a continuation of it.
