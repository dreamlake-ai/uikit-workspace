## User vs. org

Two kinds, one paint. **User** uses a 4px radius and a lighter ink fill
(`color-mix(in oklab, var(--color-ink) 10%, transparent)`); **org** uses a
2px radius and a heavier fill (`14%`) plus a slightly stronger ink for the
letters. There is no hue per user — both kinds draw from the same single
ink token, so an avatar never competes with the page's tinted accents.
