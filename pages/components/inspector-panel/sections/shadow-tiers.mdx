## Shadow tiers

Three tiers, no fourth. **Resting** is the default — the canonical
`0 1px 2px tint-1, 0 8px 24px -10px tint-3, 0 16px 40px -16px tint-2` stack
that grounds resting cards. **Lifted** is the deeper popover stack
(`0 2px 6px tint-2, 0 24px 60px -16px tint-3`) for menus, modals, anything
floating on top of the page. **Hairline** drops the shadow entirely — used
when a panel sits inside another panel and a second shadow would double up.
All three tiers compose only `--shadow-tint-{1,2,3}`; dark theme overrides
those three vars to crank the alpha so the tiers still read.
