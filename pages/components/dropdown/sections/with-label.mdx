## With a label

The optional `label` prop renders an uppercase mono caption above the control —
9px, 70% opacity, `+0.12em` tracking. Use it when the dropdown is not adjacent to
its own field label and the value text alone (`all · 38`, `main`, `running · 4`)
doesn't carry enough context. Skip it when the dropdown is part of a larger form
row that already names the field.
