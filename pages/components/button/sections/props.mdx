## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `variant` | `'ghost' \| 'primary'` | `'ghost'` | Visual treatment. Ghost is the default for inline / secondary actions; primary is the single accent-fill CTA per surface. |
| `icon` | `ReactNode` | — | Optional leading slot rendered before the label at .65 opacity. Meant for key-combo glyphs (`⌥⌘K`), not full icons. |
| `children` | `ReactNode` | — | The button label. Required. |
| `…rest` | `ButtonHTMLAttributes<HTMLButtonElement>` | — | Every standard `<button>` attribute (`onClick`, `disabled`, `type`, `aria-*`, …) flows through. |

`type` defaults to `"button"` rather than the HTML default `"submit"` — opt
into form submission explicitly with `type="submit"`.
