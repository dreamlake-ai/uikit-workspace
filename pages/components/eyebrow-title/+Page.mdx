# Eyebrow + title — mono caps eyebrow · UI title · the page-head & section-head pattern

<Lede>
  A two-line chrome atom: a mono-caps **eyebrow** label stacked above a UI
  **title**. The eyebrow is a small-caps cue (`<div>`, not a heading) — it
  positions the title without polluting the heading outline. The title itself
  renders as `<h1>`, `<h2>`, or `<h3>`, so the page outline, screen readers,
  and rehype-slug all see real semantic structure.
</Lede>
