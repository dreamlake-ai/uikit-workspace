## With a leading icon

The optional `icon` slot accepts any node — a bucket-coloured dot, a mono
glyph, an SVG. It renders ahead of the label at 80% opacity so the label keeps
priority. Pair the dot's color with the chip's tone for status chips; for
non-status chips, a mono character (`●`, `▸`, `#`) reads quietly enough that it
doesn't fight the label.
