## Styles

Three style variants. **`dot`** is the default — a static 8px circle in the
bucket hue, the right read for steady state (`ok`, `queued`, `stale`).
**`dot-pulse`** wraps the same dot in a soft 1.6s halo and is reserved for
genuinely-live state — a job ticking right now, a stream open, a heartbeat
within the last few seconds. **`bar`** swaps the circle for a 12×3 rounded
rectangle; use it inline when a dot would compete with a near-by glyph and
the indicator should read as a tag-mark instead.
