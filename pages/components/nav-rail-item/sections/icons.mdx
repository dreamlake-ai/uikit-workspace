## Icon variants

Four leading-icon choices: `none`, `folder`, `dot`, `branch`. Each sits in a
fixed 13px-wide column at muted ink, so iconed and iconless rows align
identically along the leading edge.

<NavRailItemIconsSpecimen />

- **`none`.** No glyph; the label takes the leading edge. Use for plain group
  entries — overview pages, "all", section roots.
- **`folder`.** A 13×13 Lucide-style stroke folder. Reads as "workspace" /
  "container" — projects, pipelines, environments.
- **`dot`.** A 6px filled circle in current ink. Reads as a status pip; pair
  with a tinted parent or a status-tone row to color it.
- **`branch`.** A 13×13 Lucide-style stroke git-branch glyph. Reads as a ref
  — `main`, `rc`, a deploy channel.
