# Nav-rail item — left-rail row · LeftItem shape · 5px 8px · radius · `--selected-bg` fill

<Lede>
  The atom every left-rail row is cut from. A flex row at 5px×8px padding
  and 8px radius, mono / ui mix, with a 13px icon column, a flex-1 label,
  and an optional mono trailing meta slot (count, branch, status). Three
  states: `rest`, `hover` (5% ink wash), and `active` (selected — paints
  the row with `--selected-bg` and pushes the label to weight 600). The
  in-app `LeftNav` has been painting this shape inline; this atom is the
  reusable surface other lists and menus can build on.
