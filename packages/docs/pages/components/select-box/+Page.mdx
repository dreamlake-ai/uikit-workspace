# Select Box

A compact, `options`-driven inline dropdown with a mono trigger â€” for toolbar
sort/filter controls. Opens below the trigger by default (`placement="top"`
opens upward) and closes on outside click. The trigger is `w-fit`, so it hugs its
content even inside a flex [Field](/components/field).

> For the composable trigger/value/content/item API (drop-in with the legacy
> kit's Radix Select), see [Select](/components/select).

| Prop | Type | Default | Description |

| `value` | `string` | â€” | Selected option value. |

### SelectBoxOption

| Field | Type | Description |
| --- | --- | --- |
| `value` | `string` | Unique identifier. |
| `label` | `ReactNode` | Row content (and trigger, unless `triggerLabel` is set). |
| `triggerLabel` | `ReactNode` | Compact collapsed-trigger content. Falls back to `label`. |
| `hint` | `ReactNode` | Muted right-aligned row text (e.g. a short code). |
