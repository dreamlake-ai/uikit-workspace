# Buttons — ghost & primary

Two variants. **Ghost** is the default action surface — muted ink, faint hairline,
transparent fill. **Primary** is the accent-fill CTA, one per surface. Mono
10.5px caps, 6px radius, 1px border. The same two are the only buttons every
dream.lake product reuses.
