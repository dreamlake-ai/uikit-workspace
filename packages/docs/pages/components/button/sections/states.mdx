## States

Four states per variant: `rest`, `hover`, `focus`, `disabled`. Hover and focus in the
specimen below are *forced* — the styling is set unconditionally so the row reads as a
side-by-side reference, not as something you have to mouse over.

<ButtonStates />

- **Rest.** Ghost: muted ink + faint border. Primary: accent fill + white ink.
- **Hover.** Ghost lifts to ink + 4% ink wash. Primary darkens via
  `color-mix(in srgb, var(--color-accent) 88%, var(--color-ink))`.
- **Focus.** A 2px accent ring at 50% alpha. Driven by `:focus-visible`, so keyboard
  focus rings appear without polluting click-driven focus.
- **Disabled.** Native `disabled` attribute → 40% opacity + `not-allowed` cursor. Hover
  styles short-circuit via the `not-disabled:` variant.
