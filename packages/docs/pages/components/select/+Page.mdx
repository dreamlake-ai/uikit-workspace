# Select

A compact inline dropdown for selection. Opens below the trigger by default
(`placement="top"` opens it upward), closes on outside click. Commonly used for
sort/filter controls in toolbars.

The trigger is `w-fit` — it hugs its content even inside a flex container (like a
[Field](/components/field)), so the dropdown stays anchored to the trigger
rather than a stretched parent.

Pass any glyph or node as `icon` — it renders before the label at 65% opacity.

## Alignment & forms

`align` controls which edge of the **trigger** the dropdown opens from — `'right'`
(default) suits a right-aligned toolbar control; `'left'` suits a left-aligned
control such as one inside a form Field. The label/trigger style is unchanged;
only the panel's anchor moves.

<Preview source={fieldSrc} filename="FieldSpec.tsx">
  <FieldSpec />
</Preview>

## Placement & compact trigger

For a trigger near the viewport bottom (a chat composer toolbar, a footer
control) `placement="top"` opens the panel **upward** so the options aren't
clipped. Two per-option fields keep the control tight: `triggerLabel` shows a
compact abbreviation in the collapsed trigger (falling back to `label`), while
`hint` adds a muted, right-aligned code to each dropdown row. All three are
optional — omit them and the Select behaves exactly as above.

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `value` | `string` | — | Currently selected option value. |
| `onChange` | `(value: string) => void` | — | Called when the user picks an option. |
| `options` | `SelectOption[]` | — | List of options to display. |
| `icon` | `ReactNode` | — | Optional prefix glyph rendered before the label at 65% opacity. |
| `align` | `'left' \| 'right'` | `'right'` | Edge of the trigger the dropdown panel opens from. |
| `placement` | `'top' \| 'bottom'` | `'bottom'` | Vertical side the panel opens toward. `'top'` opens upward. |
| `className` | `string` | — | Extra classes on the root wrapper. |

### SelectOption

| Field | Type | Description |
| --- | --- | --- |
| `value` | `string` | Unique identifier for the option. |
| `label` | `ReactNode` | Content rendered in the dropdown row (and the trigger, unless `triggerLabel` is set). |
| `triggerLabel` | `ReactNode` | Optional compact content for the collapsed trigger. Falls back to `label`. |
| `hint` | `ReactNode` | Optional secondary text shown right-aligned and muted in the dropdown row (e.g. a short code). |
