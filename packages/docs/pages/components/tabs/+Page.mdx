# Tabs

An animated tab switcher with two visual variants. Both share the same
controlled / uncontrolled API тАФ only the appearance changes. Content rendering
is left to the caller.

## Underline (default)

The default `underline` variant: a sliding bottom indicator under the active
tab.

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `tabs` | `Tab[]` | тАФ | Array of tab descriptors. |
| `variant` | `'underline' \| 'segment'` | `'underline'` | Visual style. |
| `value` | `string` | тАФ | Controlled active tab value. |
| `defaultValue` | `string` | `tabs[0]` | Initial tab in uncontrolled mode. |
| `onChange` | `(value: string) => void` | тАФ | Called when the user clicks a tab. |
| `onValueChange` | `(value: string) => void` | тАФ | Alias for `onChange`. |
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg'` | `'md'` | Controls text size and button dimensions across all variants. |
| `indicatorHeight` | `number` | `4` | Indicator thickness in px. `underline` variant only. |
| `className` | `string` | тАФ | Extra classes on the container. |

### Tab

| Field | Type | Description |
| --- | --- | --- |
| `value` | `string` | Unique identifier for the tab. |
| `label` | `ReactNode` | Content rendered inside the tab тАФ string, icon, or any element. |
| `count` | `number` | Count badge shown next to the label. `underline` variant only. |
| `title` | `string` | Tooltip text. Useful for icon-only segment tabs. |

## Compound API

Instead of the data-driven `tabs` prop, you can compose the tab bar and its
panels from sub-components. Pass `value` / `defaultValue` and
`onValueChange` (or `onChange`) to `Tabs`, then place `TabsList` /
`TabsTrigger` / `TabsContent` as children. Each `TabsContent` renders only
when its `value` matches the active tab.

```tsx
<Tabs defaultValue="overview" onValueChange={setTab}>
  <TabsList>
    <TabsTrigger value="overview">Overview</TabsTrigger>
    <TabsTrigger value="activity">Activity</TabsTrigger>
  </TabsList>
  <TabsContent value="overview">тАж</TabsContent>
  <TabsContent value="activity">тАж</TabsContent>
</Tabs>
```

### `TabsList`

Row container for the triggers. Accepts any `<div>` props plus `className`.

### `TabsTrigger`

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `value` | `string` | тАФ | Identifier this trigger activates. Required. |
| `className` | `string` | тАФ | Extra classes on the button. |

Also forwards native `<button>` attributes (`onClick`, `disabled`, тАж).

### `TabsContent`

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `value` | `string` | тАФ | Panel is shown only when this matches the active tab. Required. |
| `className` | `string` | тАФ | Extra classes on the panel. |
