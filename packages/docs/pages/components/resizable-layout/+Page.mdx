# ResizableLayout

A three-column drag-to-resize layout. Column widths are proportional
(flex-based) by default so the layout responds to any container size, but the
left column can be pinned to a fixed pixel width via `leftFixedPx`. Widths are
persisted to `localStorage` via `storageKey`. Each divider reveals a
cursor-following frosted pill on hover/drag; an opt-in `showToggle` prop adds
a hover-revealed collapse button.

## Demo

Drag the dividers to resize. All three columns are resizable by default. Hover
a divider to see the pill track the cursor.

## Custom gap

`gap` controls the width of the space between columns in pixels (default `24`).
Narrower gaps work well in dense layouts.

## Column toggle

Pass `showToggle` together with `onToggleLeft` / `onToggleRight` to reveal a
hover-shown collapse button on the divider. The button is opt-in — by default
the divider is just the pill.

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `left` | `ReactNode` | — | Content for the left column. |
| `middle` | `ReactNode` | — | Content for the middle column. |
| `right` | `ReactNode` | — | Content for the right column. |
| `top` | `ReactNode` | — | Optional content rendered above all columns (e.g. a toolbar). |
| `showDivider` | `boolean` | `true` | Show the cursor-following pill on dividers. The drag area is always active. |
| `showToggle` | `boolean` | `false` | Show the morphing collapse-toggle button on the divider when `onToggleLeft` / `onToggleRight` is provided. |
| `gap` | `number` | `24` | Width of the gap between columns in px. |
| `defaultWidths` | `{ left?: number; middle?: number; right?: number }` | `2/3/5` | Initial flex weights for each column. Ignored for the left column when `leftFixedPx` is set. |
| `minWidths` | `{ left?: number; middle?: number; right?: number }` | `80px` | Minimum pixel width per column, enforced during drag-resize. |
| `leftFixedPx` | `number` | — | Pin the left column to a fixed pixel width (`flex: 0 0 <px>`). The column ignores its flex ratio and never grows with the viewport; middle and right keep their flex behavior. |
| `storageKey` | `string` | — | `localStorage` key to persist column widths across sessions. |
| `leftHidden` | `boolean` | `false` | Collapse the left column. |
| `middleHidden` | `boolean` | `false` | Collapse the middle column. |
| `rightHidden` | `boolean` | `false` | Collapse the right column. |
| `leftResizable` | `boolean` | `true` | Enable drag-to-resize on the left divider. |
| `rightResizable` | `boolean` | `true` | Enable drag-to-resize on the right divider. |
| `onToggleLeft` | `() => void` | — | Toggle handler for the left divider. Only renders the toggle button when `showToggle` is true. |
| `onToggleRight` | `() => void` | — | Toggle handler for the right divider. Only renders the toggle button when `showToggle` is true. |
| `className` | `string` | — | Extra classes on the root element. |
