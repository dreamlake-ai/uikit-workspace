# Tag

Two tag styles in one component. Pick `variant` by the role the tag plays:

- **`variant="tag"` (default)** ‚Äî hashtag presentation. `#name` with an
  underline when active. Used in experiment metadata, dataset bindrs, profile
  project footers.
- **`variant="pill"`** ‚Äî boxed pill with a hairline border, fills with accent
  when active. Used for filter chips, single-select tag pickers.

## Removable tags

Hover any tag to reveal the √ó button. Clicking √ó calls `onRemove` ‚Äî wire it
to delete from your state. Hovering the √ó turns the row red and adds a
strikethrough preview.

## Active state and accent override

`active` underlines the label. By default, hover and the active underline both
use `var(--accent)`. Pass `accent` to override ‚Äî accepts a palette tone name
(`'green'`, `'red'`, ‚Ä¶) or any CSS color string. The static text stays
`var(--ink)` ‚Äî only hover and the active underline pick up the accent.

</Preview>

## Tone (palette colors)

`tone` constrains the tag to one of the design's six palette buckets and sets
BOTH the static text color AND the default accent (hover/active). Use it for
status-colored labels ‚Äî `#failed`, `#running`, `#stale`. For the rare case
where you want the static color to differ from hover, override `accent`
independently.

> **Confirmation flows** ‚Äî `Tag` fires `onRemove` immediately on √ó click. If

| Prop | Type | Default | Description |

| `name` | `string` | ‚Äî | Tag label text. |

| `active` | `boolean` | `false` | Pill fills with accent; tag underlines. |
| `removable` | `boolean` | `false` | Show a √ó button on hover that calls `onRemove`. |

### TagTone

Six semantic-palette buckets per `design.md`. Pick by **function**, not vibe.

| Tone | Hex | Description |
| --- | --- | --- |
| `'blue'` | `#23aaff` | Active / running / accent. Also the library's `--accent` token. |
| `'green'` | `#1f8f4a` | Ok / source / success. "passed", "ingest", "draft release". |
| `'amber'` | `#c0922e` | Stale / scheduled / filter. "Needs review", warnings. |
| `'red'` | `#c8513b` | Error / sink / quarantine. Failed states, destructive actions. |
| `'purple'` | `#7c5bd9` | Model / merge / human-authored. PRs, tags, code authorship. |
| `'warmGray'` | `#9c907a` | Idle / queued / placeholder. Neutral chips, "noop". |
