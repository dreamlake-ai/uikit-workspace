# Doc Site Template

A documentation site you can fork to document your own library or
project. Vike + MDX + Tailwind v4, dreamlake-v2 styling, three-column
shell, dual-theme code blocks, live previews, search palette, dark
mode. The page you are reading is rendered by the template itself —
read on to see what's in the box and how to make it yours.

## Where to next

- **[Set it up →](/getting-started)** — install, configure, replace the
  placeholder library with your own, push to Netlify.
- **[What's in the box →](/components)** — the MDX components you can
  drop into any page: `<Callout>`, `<Preview>`, fenced code blocks with
  metadata, auto-anchored headings, and a few markdown niceties.
- **[Example library page →](/library/button)** — a complete component
  doc using the template's `library/` package. Mirror this pattern for
  every component in your real library.
